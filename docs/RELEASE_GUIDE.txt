小智剪辑分类账 v0.1.0 正式版

【运行方法】
1. 将压缩包完整解压到本地文件夹，不要只单独取出 EXE。
2. 双击“小智剪辑分类账.exe”。
3. 首次启动会自动创建本地账本；点击左侧“浏览…”选择素材目录即可使用。

【系统要求】
- Windows 10/11 64 位。
- 已随程序附带 .NET 桌面运行环境，无需另装 .NET。

【FFmpeg（可选）】
- 不安装 FFmpeg 仍可管理素材、状态、备注和项目。
- 如需代表帧、悬停扫片和兼容预览，请在软件右上角设置 ffmpeg.exe；
  也可以把 ffmpeg.exe 和 ffprobe.exe 放入程序目录下的 tools 文件夹。

【数据位置与安全】
- 项目、素材状态、备注和设置保存在：%LOCALAPPDATA%\XiaoZhiVideoLedger
- 原视频不会被程序移动、重命名或删除。
- 更新程序时可直接替换程序文件夹，本地账本不会随程序文件夹被删除。
- 如需迁移到另一台电脑，请同时备份上述数据目录和你的原素材目录。

【分享注意】
- 请分享完整 ZIP，不要只发送“小智剪辑分类账.exe”。
- shared、tools、DLL 和 hostfxr.dll 均属于程序运行所需或预留的组成部分。
